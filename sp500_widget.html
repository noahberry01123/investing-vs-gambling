Data & method. S&P 500 (^GSPC) year-end closing prices via Yahoo Finance, 1927–2025; annual returns are year-over-year changes in the year-end close (price return, excluding dividends). The chart rebases the index to 100 at the start of the selected year and compounds each year's return.

Dividends. The Dividends toggle adds an assumed constant 3% annual dividend yield, reinvested (each year's growth is multiplied by 1.03). This is a simplification: the S&P 500's actual yield was higher (often 4–6%) for much of the 20th century and below 2% in recent years, so it understates dividends for early windows and slightly overstates them for recent ones.

Inflation. Real returns deflate by U.S. CPI-U (BLS series CUUR0000SA0, December values) — the same consumer price index that FRED publishes seasonally adjusted as CPIAUCSL. CPIAUCSL itself begins in 1947, but the underlying CPI-U extends back to 1913, so inflation-adjusted returns are available across the entire 1928–2025 range. (If CPI were unavailable for a year, the widget would note that real returns are only available from 1947 onward.)
